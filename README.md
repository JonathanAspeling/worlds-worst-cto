# The CTO Leadership Self-Assessment

An interactive web application for technology leaders to honestly assess their leadership tendencies under pressure.

## Overview

This is a brutally honest, 20-question assessment designed to surface how CTOs and technology leaders actually think about leadership when systems fail, timelines slip, engineers disagree, and executives demand simple answers to complex problems.

The assessment is written like a Cosmopolitan quiz for people who run technology at scale. Every option is phrased to sound reasonable, responsible, and professionally defensible. That's the point. The wrong answers are rarely stupid - they are seductive.

## Features

- **Landing Page**: Comprehensive explanation of the assessment with clear scoring indicators
- **Interactive Question Flow**: 20 questions presented one at a time with smooth transitions
- **Progress Tracking**: Visual progress indicator showing current question and total
- **Automatic Scoring**: Instant calculation of results across three leadership categories
- **Detailed Results**:
  - Score summary with percentages for each category
  - Personalized interpretation based on results
  - Complete breakdown showing all questions with your answers highlighted
  - Full explanations for each option (what makes it attractive vs. what it creates)
- **Mobile-First Design**: Fully responsive layout optimized for all devices
- **No Build Required**: Pure HTML, CSS, and JavaScript - just open index.html

## Usage

### Running Locally

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Click "Start Assessment"
4. Answer all 20 questions honestly
5. Review your results and detailed breakdown

No server, no build process, no dependencies. Just double-click `index.html`.

### Deploying to a Server

Simply upload all files to your web server and point to `index.html`. The application is entirely client-side and works on any standard web hosting.

## About the Assessment

The most dangerous leaders are not incompetent. They are reassuring.

This assessment helps technology leaders recognize the difference between leadership that looks good in the moment and leadership that builds durable, resilient organizations. Each question presents four options that all sound professionally defensible - because that's how these decisions actually present themselves in the real world.

Answer honestly. Not as the leader you describe in interviews. As the leader you become when pressure arrives and incentives collide with reality.
